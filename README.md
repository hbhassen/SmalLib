# SmalLib

Librairie SAML minimaliste conforme aux exigences décrites dans `SPECIFICATION.md`, `ttd.md` et `plantest.md`.

## Documentation
- Le panorama complet des modules, la traçabilité des exigences (`TC-*`) et les consignes d'intégration sont détaillés dans `docs/LIBRARY.md`.
- Les diagrammes UML et la spécification technique détaillée se trouvent respectivement dans `umlConception.md` et `SPECIFICATION.md`.

## Tests
Exécuter la suite unitaire et d'intégration simplifiée :

```
mvn test
```

Les méthodes de test incluent les identifiants `TC-*` pour refléter le plan de tests officiel.
